# Python Programming Internship Tasks

## Overview

This repository contains the Python Programming Internship Tasks completed as part of the internship program.

The project includes tasks from Easy, Medium, and Hard difficulty levels to demonstrate understanding of Python fundamentals, problem-solving, automation, file handling, and application development.

## Tasks Completed
### Easy Level Tasks
#### 1. Simple Calculator
A calculator that performs:

* Addition
* Subtraction
* Multiplication
* Division

Concepts Used:

* User Input
* Conditional Statements
* Arithmetic Operators
#### 2. Number Guessing Game

A game where the computer generates a random number and the user attempts to guess it.

Features:

* Random number generation
* Hint system (Too High / Too Low)
* Loop until correct answer

Concepts Used:

* Loops
* Conditions
* Random Module

#### 3. Word Count Program

Counts the number of words entered by the user.

Concepts Used:

* String Handling
* split() Function
* User Input

### Medium Level Tasks

#### 4. Password Generator

Generates secure random passwords using:

* Letters
* Numbers
* Special Characters

Concepts Used:

* Random Module
* String Module
* Lists
#### 5. Contact Book Program
A simple contact management system.

Features:

* Add Contact
* View Contacts
* Search Contact

Concepts Used:

* Dictionaries
* Functions
* Menu Driven Programming

### Hard Level Task

#### 6. Command Line To-Do List Application

Features:

* Add Tasks
* View Tasks
* Delete Tasks
* Save Tasks During Runtime

Concepts Used:

* Lists
* Functions
* Loops
* CRUD Operations
